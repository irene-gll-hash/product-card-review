# Experiments

Каталог конфигураций и результатов экспериментов по проверке карточек товаров.

Каждое направление содержит:

- `configs/` — воспроизводимые параметры моделей;
- `results/` — метрики, логи и артефакты запусков;
- `comparison.md` — сводное сравнение моделей.

Конфиги используют пути относительно корня проекта. Датасет и сгенерированные
артефакты в этом каталоге не хранятся.
